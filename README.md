# customer-behavior-analysis
data analytics project showcasing of customer behavior analysis using Python, SQL, Power BI
📊 Data Analysis Project
📌 Overview

This project demonstrates an end-to-end data analysis workflow, starting from data loading and exploration to visualization and dashboard creation. The goal is to extract meaningful insights from raw data using Python, SQL, and Power BI.

📂 Dataset
The dataset used in this project contains structured data related to customer transactions.
It includes features such as:
Customer details
Product information
Purchase amounts
Review ratings
Shipping types

🛠️ Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn)
PostgreSQL (for querying and data analysis)
Power BI (for dashboard and visualization)
Jupyter Notebook (for development and analysis)

🔄 Project Workflow
1. Data Loading
Imported dataset using Pandas
Checked structure using .head(), .info(), .describe()
2. Exploratory Data Analysis (EDA)
Analyzed distributions and relationships
Identified trends, patterns, and anomalies
Visualized data using charts (bar plots, histograms, scatter plots)
3. Data Cleaning
Handled missing values
Removed duplicates
Corrected data types
Standardized column names
4. SQL Analysis (PostgreSQL)
Connected Python with PostgreSQL
Performed queries such as:
Top-selling products
Average ratings
Discount analysis
Shipping type comparisons
5. Data Visualization (Power BI)
Imported cleaned data into Power BI
Created interactive dashboards
Added filters, slicers, and KPIs

📊 Dashboard
The Power BI dashboard includes:
Sales performance overview
Product-wise analysis
Customer insights
Shipping and discount trends

📈 Results & Insights
Identified top-performing products
Found relationship between discounts and sales
Analyzed customer behavior patterns
Compared shipping methods and their impact

▶️ How to Run the Project
Step 1: Clone the Repository
git clone <your-repository-link>
cd project-folder
Step 2: Install Required Libraries
pip install pandas numpy matplotlib seaborn psycopg2
Step 3: Run Python Scripts
Open Jupyter Notebook
Execute data loading, EDA, and cleaning steps
Step 4: Setup PostgreSQL
Create database
Import dataset
Run SQL queries
Step 5: Open Power BI Dashboard
Load dataset into Power BI
Open .pbix file to view dashboard

This project showcases a complete data analysis pipeline using industry-standard tools. It highlights the ability to clean, analyze, query, and visualize data to generate actionable insights.
